## To Start 🚀
* Clone the repo `https://github.com/Jony-Jas/amazon-clone-web3`
* Run `npm install` in both root and site folder
* Compile and Deploy the Amazon.sol contract
* Replace the CONTRACT_ADDRESS in `next.config.js`
* Run `npm run dev`
* Enjoy :)
